# Bundled CC0 starter sounds

These `.wav` files are **original, procedurally-synthesized** sound effects generated by
[`scripts/generate_starter_sfx.py`](../../../../scripts/generate_starter_sfx.py) and dedicated to the public
domain under **Creative Commons CC0 1.0**. They can be redistributed with foley-forge and used commercially
with no attribution.

They are intentionally simple placeholder foley — enough to make the pipeline produce a real, editable
timeline out of the box. For production sound, add your own packs:

```bash
foley-forge index-sfx /path/to/your/sfx -o my_manifest.json
# then reference it via [sfx] libraries in your config, or point the library loader at the folder.
```

Truly bundle-able free sources to grow this pack: **Kenney.nl** (CC0), CC0-filtered **OpenGameArt**, and
CC0 files from **Freesound**. Do **not** add Sonniss/Zapsplat/Mixkit/Pixabay/BBC files here — they forbid
redistribution as a raw library (see the repository `NOTICE.md`).

`manifest.json` records per-file id, tags, license, and provenance.
